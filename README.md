# ICI_irAE
Code to reproduce the essential analyses in the Mars et al ICI irAE paper.

The code files are numbered with sequential numbers but do not have to be run in that order.

Every omics layer has its own .Rmd / .R file who independently also call 1_ICI_cohort_description.Rmd  to load the clinical metadata.

Data used to reproduce the essential analyses of the ICI irAE cohort are shared in Supplementary Table 1 which is called in the code files.
